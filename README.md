# FVPVM

## 0x00 Why FVPVM?

我们的目标：使用Python / C++ 编写 FVP VM 的自由实现

## 现在的工具：

- vm.py
Python 编写的 FVP VM 实验性实现 **（目前处于pre-alpha阶段）**
- frida_script
基于 FRIDA 编写的脚本，能使得FVP原版引擎实时反汇编&输出栈信息
- ida_plugin
IDA Pro 的插件，使得 IDA Pro 可以加载&反汇编 FVP VM 的二进制文件

## Roadmap
